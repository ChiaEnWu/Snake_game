## 設計理念

Evolved Snake 將經典貪吃蛇重新想像為一款霓虹賽博龐克風格的技能驅動街機遊戲。核心概念是保留原作即開即玩的直覺手感，同時透過技能經濟系統注入策略深度。

視覺風格源自 synthwave 與賽博龐克美學——暗色背景配上霓虹綠、粉紅與紫色的發光元素，碰撞時的粒子爆發效果，以及脈動的格線讓每次移動都充滿回饋感。20×20 的棋盤提供足夠的戰術空間而不顯得稀疏。

技能系統是本作的核心：四種技能（加速、減速、無敵、雙倍分數）透過遊戲中隨機生成的金色食物解鎖。由於技能有冷卻時間且需手動觸發（按鍵 1-4），玩家必須判斷每個技能的使用時機——在開闊通道中加速衝刺、穿牆逃離死路、或在 combo 疊高時開啟雙倍分數。這創造了傳統貪吃蛇所沒有的風險報酬權衡。

連續進食會疊加 Combo 倍率，而所有音效皆透過 Web Audio API 程式生成——吃食物、啟動技能、Combo 連擊到死亡，每一步都有即時的聽覺回饋。最終的成果是一款更像現代街機遊戲、而非瀏覽器時代遺物的貪吃蛇。

## 和原版 Snake 的最大差異

| 項目 | 經典貪吃蛇 | Evolved Snake |
|------|-----------|---------------|
| 技能 | 無 | 4 種可解鎖技能（1–4 鍵手動使用） |
| 金色食物 | — | 隨機出現，吃了獲得隨機技能 |
| Combo 系統 | — | 連續進食疊加分數倍率 |
| 牆壁碰撞 | 直接死亡 | 預設死亡，無敵技能可穿牆 |
| 視覺風格 | 單色或簡約 | 霓虹賽博龐克、粒子特效、發光 |
| 音效 | 無或簡單嗶聲 | Web Audio API 合成音效 |
| 語言 | 單一 | 3 種語言（繁體中文 / EN / 日本語）|
| 操作 | 方向鍵 | 方向鍵 + WASD，P 暫停 |

## 怎麼玩

**操作：** 方向鍵或 WASD 控制移動。按 **P** 暫停／繼續。按 **1–4** 使用已解鎖的技能。

目標仍然是吃食物讓蛇變長，但 Evolved Snake 增加了幾個改變每一步決策的層次：

- **金色食物**會不定期出現——吃掉它可解鎖一個隨機技能。技能為一次性使用，有冷卻時間，請謹慎選擇時機。
- **Combo** 隨著連續進食而增加。Combo 越高，每顆食物的分數越多。死亡時 Combo 歸零。
- **四種技能**讓你可以打破規則：在空曠處加速穿越、在狹窄角落減速微調、穿牆繞過障礙、或開啟雙倍分數衝高分。
- **預設不會穿牆**——碰到牆壁即死亡，除非無敵技能啟用。這改變了你在邊緣追食物時的風險判斷。

最大的不同在於你不只是要閃避自己——你同時在管理資源（技能）、留意金色食物的出現時機、以及決定何時要積極進攻疊 Combo、何時該穩紮穩打。這是一代每隔幾秒就需要做出有意義抉擇的貪吃蛇。
